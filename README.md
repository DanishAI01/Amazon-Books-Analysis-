# Amazon-Books-Analysis-
 Amazon Books Analysis – Exploratory Data Analysis of Amazon bestselling books (2009–2019) using Python to uncover trends in genres, top authors, and reader preferences through data visualization.
# 📚 Amazon Books Analysis

## 📌 Project Description

This project performs **Exploratory Data Analysis (EDA)** on Amazon Bestselling Books data to identify trends in book genres, popular authors, ratings, reviews, and pricing patterns.
The analysis helps understand reader preferences and bestseller trends on Amazon from **2009 to 2019**.

---

## 🎯 Objectives

* Clean and preprocess the dataset
* Analyze Fiction vs Non-Fiction book trends
* Identify top authors and bestselling books
* Explore yearly publishing patterns
* Visualize insights using graphs and charts

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 📂 Dataset Information

The dataset contains:

* Book Name
* Author
* User Rating
* Reviews
* Price
* Year
* Genre

---

## 📊 Analysis Performed

* Data Cleaning & Handling Missing Values
* Duplicate Data Removal
* Genre Distribution Analysis
* Top Authors Analysis
* Year-wise Bestseller Trends
* Data Visualization using Charts

---

## 📈 Key Insights

* Non-Fiction books frequently dominate bestseller lists.
* Some authors consistently appear across multiple years.
* Ratings and reviews strongly influence bestseller rankings.
* Genre popularity changes over time.

## 🚀 How to Run the Project

1. Clone this repository:

   ```bash
   git clone https://github.com/your-username/amazon-books-analysis.git
   ```

2. Navigate to the project folder:

   ```bash
   cd amazon-books-analysis
   ```

3. Install required libraries:

   ```bash
   pip install pandas numpy matplotlib seaborn
   ```

4. Open Jupyter Notebook:

   ```bash
   jupyter notebook
   ```

5. Run **amazon Books Analysis.ipynb**
